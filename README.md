**Repository:** [InsightForge](https://github.com/rafaelgutierres049/InsightForge)

InsightForge is an Azure-native document intelligence and RAG platform designed to extract, process, index, and answer questions over private documents using Azure AI services and LLMs.

**Key capabilities:**

- Document ingestion and extraction using Azure AI Document Intelligence
- Chunking and preprocessing pipeline
- Embedding generation with Azure OpenAI
- Hybrid retrieval with Azure AI Search
- Secure document storage with Azure Blob Storage
- FastAPI backend for orchestration
- Streamlit interface for user interaction
- Grounded answer generation with GPT-based models

**Tech stack:**  
Python · FastAPI · Azure OpenAI · Azure AI Search · Azure Document Intelligence · Azure Blob Storage · Streamlit · RAG
